Effortlessly display the current date with precision using a streamlined shortcode

== Description ==

Effortlessly enhance your website's functionality by integrating a streamlined shortcode that seamlessly displays the current date with precision. With this intuitive implementation, users can effortlessly embed the current date wherever necessary, ensuring accuracy and reliability across your web platform. This optimized solution not only simplifies the process of showcasing date information but also adds a layer of sophistication to your website's user experience. Experience the convenience of dynamic date presentation while maintaining the highest standards of performance and reliability.

== Installation ==

There is no prerequisites for installing **Current Date**. You can follow the steps below to complete the installation.

= Automatic Installation =
1.  Go to `Plugins > Add New` screen from your WordPress website dashboard.
2.  Search for `Current Date`.
3.  Install and activate the plugin, that's it!
